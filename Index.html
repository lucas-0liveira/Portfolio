PROJETOS DESENVOLVIDOS

  • Contador para Exame

    Este projeto é uma aplicação web desenvolvida com HTML, CSS e JavaScript, que permite o controle de exames médicos, como raio-X, mamografia e tomografia. O usuário pode selecionar o tipo de exame e clicar em "Incrementar" para registrar cada procedimento realizado. A aplicação também oferece a opção de gerar um relatório em PDF com o total de exames incrementados por tipo, assim como o total mensal acumulado, facilitando o monitoramento e a impressão de registros médicos.

  • Play Music

    Este projeto é um reprodutor de música personalizado, desenvolvido com JavaScript, HTML e CSS. A aplicação permite ao usuário ouvir músicas, favoritar suas músicas, alternar entre faixas e reproduzir músicas de forma aleatória.

  • Cardápio Lanches

    Este projeto foi criado para oferecer uma experiência de cardápio de lanches personalizável, permitindo filtrar opções rapidamente através de funcionalidades avançadas em JavaScript (ForEach, Map, Reduce e Filter), HTML e CSS. Além disso, é possível alternar no site para modo claro e modo escuro.

  • Dev Sorteio

    O projeto permite ao usuário inserir os valores inicial e final do intervalo, e com um clique, um número aleatório é gerado e exibido na tela. O design foi elaborado utilizando CSS para proporcionar uma interface amigável, enquanto a lógica do sorteio e a interação foram implementadas em JavaScript.

  • Convert Money

    O projeto utiliza JavaScript para calcular as conversões, HTML para estruturar a interface e CSS para um design intuitivo e moderno. Além disso, você pode inserir o valor desejado na moeda atual, e o sistema calcula de forma automática o quanto ficará na moeda de destino. Uma ferramenta prática, interativa e precisa para facilitar conversões monetárias
